## Why Bedstack?

- **Unmatched Speed**: Powered by Bun's high-performance runtime.
- **End-to-End Typesafety**: Type information flows from the database schema to the API response.
- **Developer Experience**: Minimal boilerplate, instant feedback with Bun's watch mode.
- **Production Ready**: Built on the solid foundation of the RealWorld specification.
